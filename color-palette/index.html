컬러무드 (ColorMood) 소개

감성 무드를 고르거나 색 이론 규칙을 적용해 어울리는 배색을 즉시 만들어 볼 수 있는 컬러 팔레트 도구입니다. 사진을 업로드하면 이미지에서 대표 색을 추출할 수도 있어, 디자이너뿐 아니라 콘텐츠를 만들거나 인테리어·패션 코디를 고민하는 누구에게나 유용합니다.

이렇게 사용하세요

  1. 무드 선택, 색 이론, 이미지 색 추출 중 원하는 생성 방식을 고릅니다.
  2. 생성된 5가지 색상을 팔레트로 확인합니다.
  3. 마음에 드는 색은 자물쇠 아이콘으로 고정하고 스페이스바로 나머지를 새로 생성합니다.
  4. 글자와 배경 조합의 명암 대비(WCAG 기준)를 확인해 접근성을 검토합니다.
  5. HEX 값 복사, CSS 변수, Tailwind 설정, 또는 이미지 파일로 내보냅니다.

활용 팁

  • 마음에 드는 색 하나를 고정해두고 나머지를 계속 돌리면 원하는 조합을 훨씬 빠르게 찾을 수 있습니다.
  • 웹이나 앱 UI에 쓸 배색이라면 명암 대비 검사에서 AA 이상을 통과하는지 반드시 확인하세요.
  • 이미지 색 추출 기능을 활용하면 좋아하는 사진이나 브랜드 로고에서 자연스러운 배색 힌트를 얻을 수 있습니다.

자주 묻는 질문

Q. 업로드한 사진은 서버에 저장되나요?

이미지 처리는 브라우저 내에서만 이루어지며 서버로 전송되지 않습니다. 개인 사진을 안심하고 사용할 수 있습니다.

Q. WCAG 명암 대비 검사는 무엇인가요?

웹 접근성 지침(WCAG)에 따라 텍스트와 배경색 간의 명암 비율을 계산합니다. AA 기준(4.5:1 이상)을 통과하면 시각적으로 읽기 편한 조합입니다.

Q. 만든 팔레트를 나중에 다시 볼 수 있나요?

저장한 팔레트는 브라우저에 보관되어 다음 방문 시에도 히스토리에서 찾아볼 수 있습니다.
