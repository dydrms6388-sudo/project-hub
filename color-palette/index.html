컬러무드 (ColorMood) 소개

감성 무드를 선택하거나 색 이론 규칙을 적용해 어울리는 배색을 즉시 만들어 볼 수 있는 컬러 팔레트 도구입니다. 사진을 올려 대표 색을 추출하는 기능도 있어, 디자이너뿐 아니라 SNS 콘텐츠를 만들거나 인테리어·패션 코디를 고민하는 누구에게나 유용합니다.

알아두면 좋은 배경지식

색은 보는 사람의 감정과 인식에 직접적인 영향을 주기 때문에 디자인에서 배색은 매우 중요한 요소입니다. 색 이론에서는 색상환을 기반으로 보색(정반대 색), 유사색(인접한 색), 3색 조화(120도 간격) 등의 규칙으로 조화로운 배색을 만드는 방법을 설명합니다. 웹 접근성 분야에서는 텍스트와 배경 사이의 명암 대비 비율을 WCAG 지침으로 정해 두어, 4.5:1 이상(AA)이어야 시각적으로 읽기 편하다고 봅니다. 일반인도 직관적으로 활용할 수 있도록 무드 기반 생성 방식을 함께 제공하면, 색 이론 지식 없이도 원하는 분위기에 맞는 배색을 얻을 수 있습니다.

이렇게 사용하세요

  1. 무드 선택, 색 이론, 이미지 색 추출 세 가지 생성 방식 중 하나를 고릅니다.
  2. 무드 모드에서는 원하는 감성 키워드를 선택하고, 색 이론 모드에서는 조화 방식을 고릅니다.
  3. 생성된 5가지 색상 팔레트를 확인합니다.
  4. 마음에 드는 색은 자물쇠 아이콘으로 고정한 뒤 스페이스바로 나머지를 새로 생성합니다.
  5. WCAG 명암 대비 검사로 텍스트·배경 조합의 가독성을 확인합니다.
  6. HEX 값 복사, CSS 변수, Tailwind 설정 스니펫, 또는 PNG 이미지 파일로 내보냅니다.

이럴 때 유용해요

  • 웹사이트나 앱 UI 색 시스템을 잡을 때, 브랜드 컬러를 고정하고 나머지 보조색을 빠르게 탐색하는 데 유용합니다.
  • SNS 피드 통일감을 위해 대표 사진에서 색을 추출한 뒤 같은 톤으로 게시물 배경이나 그래픽을 맞출 때 활용할 수 있습니다.
  • 인테리어 색 조합을 고민할 때, '차분한' 또는 '노을' 같은 무드 키워드로 비슷한 계열의 배색을 빠르게 시각화해 볼 수 있습니다.

활용 팁

  • 마음에 드는 색 한두 개를 자물쇠로 고정한 뒤 나머지를 계속 새로 굴리면 원하는 조합을 훨씬 빠르게 찾을 수 있습니다.
  • UI 디자인에 쓸 배색이라면 WCAG 명암 대비 검사에서 AA(4.5:1 이상)를 통과하는지 반드시 확인하세요.
  • 이미지 색 추출 시 색이 단순한 로고보다 명도 차이가 있는 풍경 사진을 쓰면 더 풍부한 팔레트가 나옵니다.
  • 오늘의 색 기능으로 매일 새로운 색을 발견하는 재미도 있습니다.

주의사항

  • 이미지 색 추출 결과는 알고리즘 기반 추정이며, 사진의 조명 조건에 따라 실제 인쇄 색과 차이가 날 수 있습니다.
  • HEX 값은 화면 표시 기준이므로 인쇄나 섬유 제작에 사용할 경우 CMYK 또는 팬톤 색상 코드로 변환하는 추가 작업이 필요합니다.
  • WCAG AA 통과 여부는 참고 기준이며, 실제 접근성 검토는 다양한 화면과 환경에서 직접 확인하는 것이 좋습니다.

자주 묻는 질문

Q. 업로드한 사진은 서버에 저장되나요?

이미지 처리는 브라우저 내 canvas 기술로만 이루어지며 서버로 전송되지 않습니다. 개인 사진이나 회사 로고를 안심하고 업로드할 수 있습니다.

Q. WCAG 명암 대비 검사는 어떻게 활용하나요?

텍스트 색과 배경 색을 선택하면 두 색 사이의 명암 비율을 계산합니다. AA 기준(4.5:1 이상)을 통과하면 대부분의 환경에서 읽기 편한 조합으로 볼 수 있습니다.

Q. 만든 팔레트를 나중에 다시 볼 수 있나요?

저장한 팔레트는 브라우저(localStorage)에 보관됩니다. 히스토리에서 이전에 만든 팔레트를 다시 불러올 수 있습니다.

Q. CSS 변수나 Tailwind 스니펫 내보내기는 어떻게 사용하나요?

내보내기 버튼을 누르면 팔레트 색상이 CSS 변수 또는 Tailwind 설정 형식의 코드로 변환됩니다. 복사해서 프로젝트 파일에 바로 붙여 넣어 사용할 수 있습니다.

Q. 색 이론 조화 방식의 차이는 무엇인가요?

보색은 색상환 반대 편 색으로 강한 대비, 유사색은 인접한 색으로 자연스러운 통일감, 3색 조화는 120도 간격으로 균형 잡힌 활기, 단색은 같은 색의 명도 변화로 세련된 느낌을 줍니다.
