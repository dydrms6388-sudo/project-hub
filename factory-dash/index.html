팩토리 대시보드 소개

매일 새로 만들어지는 서비스들을 한 곳에서 현황과 진행 상황을 한눈에 볼 수 있는 운영 전용 도구입니다. 각 서비스에 관심 표시를 해두면 다음 개발 우선순위에 반영되어, 흩어진 아이디어를 체계적으로 관리하고 개발 흐름을 유지할 수 있습니다. 목표 달성률과 메모 기능을 통해 프로젝트 전체 맥락을 놓치지 않고 이어갈 수 있습니다.

이렇게 사용하세요

  1. 대시보드에 접속해 현재까지 생성된 서비스 목록을 확인하세요.
  2. 각 서비스 카드에서 현재 상태와 기본 정보를 살펴보세요.
  3. 우선 개발하고 싶은 서비스에 관심 표시를 해두세요.
  4. 메모 기능으로 각 서비스별 개선 아이디어나 이슈를 기록하세요.
  5. 전체 목표 진행률을 확인하고 다음 작업 방향을 정하세요.

활용 팁

  • 관심 표시는 한 번에 여러 개에 하는 것보다 진짜 우선순위 1~3개에 집중해야 실제 개발 반영 효과가 높아집니다.
  • 메모는 짧고 구체적인 액션 아이템 위주로 남겨두면 나중에 다시 봤을 때 바로 실행하기 좋습니다.
  • 주기적으로 완료된 서비스의 상태를 업데이트하면 전체 진행률이 현실적으로 유지됩니다.

자주 묻는 질문

Q. 관심 표시를 하면 자동으로 개발이 시작되나요?

관심 표시는 다음 개발 실행 시 우선 순위를 높이는 신호로 작동합니다. 실제 개발 착수는 별도 확인 후 진행됩니다.

Q. 대시보드 데이터는 어디에 저장되나요?

서비스 목록과 플래그 정보는 Neon 데이터베이스와 연동되어 저장됩니다.

Q. 서비스 목록은 얼마나 자주 업데이트되나요?

새로운 서비스가 생성될 때마다 목록이 업데이트됩니다.
