스도쿠 브레인 소개

로그인 없이 바로 즐기는 무료 온라인 스도쿠. 검증된 유일해 퍼즐 생성 엔진(백트래킹 solver로 해답 유일성 보장)으로 쉬움(단서42)·보통(34)·어려움(29)·고수(25) 4단계 난이도를 즉석 생성. 핵심 리텐션 훅=오늘의 도전(날짜 시드 결정론으로 전국 동일 문제, 매일 0시 KST 갱신)+연속 출석 스트릭(손실회피)+난이도별 최고 기록. 게임 기능: 셀 선택 시 같은 줄·열·박스+동일 숫자 하이라이트, ✏️메모(펜슬마크) 모드, 💡힌트(단일후보 우선 공개), 되돌리기(히스토리 200), 틀린 숫자 자동 표시(정답 대조)·중복 충돌 표시, 실수 카운트, 일시정지, 이어하기(자동저장), 키보드(숫자·방향키·백스페이스) 지원, 숫자패드 완성 숫자 자동 디밍. 무백엔드·무가입(기록 전부 localStorage, 서버 전송 없음). 큐 잔여 전부 금융이라 비금융 발산 선정(게임 카테고리 로테이션).
