저작권 안내

기사 본문·이미지는 수집·게재하지 않습니다. 전문은 각 카드의 원문 링크에서 확인하세요.

오늘의 카드뉴스 소개

오늘의 카드뉴스는 국내 언론사들이 공개 배포하는 공식 RSS 피드에서 매일 아침 7시(한국 시간) 무렵 헤드라인을 자동 수집해 보여주는 데일리 브리핑 페이지입니다. 수집 파이프라인은 GitHub Actions 크론 작업으로 돌아가며, 기사에서 가져오는 정보는 제목·출처(언론사명)·원문 링크·발행 시각 네 가지뿐입니다. 기사 본문이나 사진은 어떤 형태로도 저장하거나 게재하지 않고, 내용이 궁금하면 카드의 '원문 보기' 링크로 해당 언론사 사이트에 이동해 읽는 구조입니다. 사람이 기사를 고르거나 순서를 바꾸는 편집 개입 없이 피드의 최신 순서를 그대로 따르며, 주요·경제·IT과학·문화연예 네 갈래로만 나눠 정리합니다. 큰 타이포그래피 카드 형식이라 아침에 30초만 훑어도 오늘 흐름이 잡히고, 각 카드는 1080×1080 정사각 PNG로 저장해 인스타그램·X 같은 SNS에 바로 올릴 수 있습니다. 날짜별 아카이브가 남아 있어 지난 날짜의 브리핑도 이전/다음 버튼으로 넘겨볼 수 있습니다.

이렇게 사용하세요

  1. 상단 카테고리 탭(주요·경제·IT과학·문화연예)을 눌러 관심 분야 헤드라인을 봅니다.
  2. 흥미로운 기사는 카드의 '원문 보기 →'를 눌러 언론사 원문을 새 창으로 읽습니다.
  3. 공유하고 싶은 카드는 'PNG 저장'을 눌러 1080×1080 이미지로 내려받습니다.
  4. '오늘 카드 전체 저장'은 주요 카테고리 상위 5개를 한 번에 순차 저장합니다.
  5. '텍스트 브리핑 복사'로 카테고리별 톱3 제목을 복사해 단톡방·커뮤니티에 붙여넣으세요.
  6. '← 이전 날짜'로 지난 브리핑을 아카이브에서 다시 볼 수 있습니다.

이럴 때 유용해요

  • 출근길에 오늘 헤드라인만 빠르게 훑고 싶을 때
  • 단톡방·커뮤니티에 아침 뉴스 요약을 공유하는 습관이 있을 때
  • SNS 계정에 올릴 깔끔한 헤드라인 카드 이미지가 필요할 때
  • 특정 날짜에 어떤 뉴스가 있었는지 아카이브로 되짚어 볼 때

자주 묻는 질문

Q. 뉴스는 언제, 어떻게 갱신되나요?

매일 아침 7시(한국 시간) 무렵 GitHub Actions 크론 작업이 언론사 공식 RSS 피드를 읽어 제목·출처·원문 링크·발행 시각만 추출해 정적 JSON 파일로 저장합니다. 이 페이지는 그 JSON을 불러와 카드로 보여줄 뿐이며, 사람이 손으로 고르거나 편집하지 않습니다. 헤더의 '업데이트' 시각이 마지막 수집 시각입니다.

Q. 기사 출처 정책은 어떻게 되나요?

언론사가 구독·재배포를 위해 스스로 공개한 공식 RSS 피드만 수집원으로 사용합니다. 모든 카드에 언론사 이름이 출처로 표기되고, '원문 보기' 링크는 해당 언론사 사이트의 기사 원문으로 새 창에서 연결됩니다. 특정 언론사를 우대하거나 제외하는 편집 개입은 없습니다.

Q. 저작권 문제는 없나요?

기사 본문·사진·이미지는 일절 수집하거나 게재하지 않습니다. 제목·출처·원문 링크만 표시해 원문으로 트래픽을 돌려보내는 링크 큐레이션 방식이며, 모든 기사의 저작권은 각 언론사에 있습니다. 언론사의 요청이 있으면 해당 피드는 수집원에서 제외합니다.

Q. 저장한 카드 PNG는 어디에 써도 되나요?

개인 SNS·커뮤니티 공유용입니다. 카드에는 제목과 출처 표기만 담깁니다. 게시할 때 카드 하단의 출처 표기와 워터마크를 지우지 말고, 가능하면 원문 링크를 함께 올려 주세요. 출처를 가리거나 상업 홍보물로 재가공하는 사용은 삼가야 합니다.

※ 이 페이지는 헤드라인 큐레이션(정보·재미용)이며 어떤 기사에 대한 사실 확인·평가·논평을 제공하지 않습니다. 의료·법률·금융·투자 판단의 근거로 사용하지 마세요.
