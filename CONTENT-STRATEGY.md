# 콘텐츠 기획 기준 (Content Strategy Gate)

> 팩토리에서 **새로 만드는 모든 콘텐츠/서비스**가 기획 단계에서 충족해야 하는 3대 기준.
> 이 문서는 아이디어 파이프라인의 **생성 게이트**다. 세 기준을 모두 통과(✅)하지 못한 아이디어는
> 개발(만족도 루프)로 내려보내지 않는다. — 연결: [`FACTORY-LOOP.md`](../2026-06-20-factory-dash/FACTORY-LOOP.md) 티어3 "창작 무한"(#12 창작 게이트), [`AGENT-TASKS.md`](../2026-06-20-factory-dash/AGENT-TASKS.md).

판정: 각 기준의 체크리스트에서 **필수 항목을 모두** 만족해야 그 기준 ✅. 세 기준 모두 ✅ → 통과.

---

## 기준 1 — 듀얼 모드 (개인 + 그룹 둘 다)

혼자 써도 가치가 있고(개인화), 동시에 남과 함께/겨뤄/공유하는 레이어(그룹)를 **둘 다** 가진다.
한쪽만 있으면 탈락. "공유 버튼만 달린 개인 도구"는 그룹으로 인정하지 않는다.

**(a) 개인 레이어** — 다음 중 1개+
- 저장되는 진행/기록 (스트릭·로그·히스토리)
- 맞춤 추천·개인화 결과
- 나만의 컬렉션·아카이브

**(b) 그룹 레이어** — 다음 중 1개+
- 결과 공유(공유 카드 이미지/링크) — *단독으로는 약함, 아래 중 하나와 결합 권장*
- 친구 대결·랭킹·리그
- 그룹방·챌린지·시즌
- 협업(같이 만드는 무엇)
- 커뮤니티 피드(익명 글·응원)

### ✅ 체크리스트 (기준 1)
- [ ] (필수) 개인 레이어가 **로그인 없이도** 동작하는 개인 가치 1개+ 있다
- [ ] (필수) 그룹 레이어가 단순 공유버튼을 넘어 **상호작용**(대결/랭킹/방/피드 중 1+) 한다
- [ ] (필수) 개인↔그룹이 **연결**된다 (내 기록이 랭킹에 들어가는 등, 두 모드가 따로 놀지 않음)
- [ ] 그룹 레이어가 **무가입/익명**으로도 최소 참여 가능 (진입장벽 ↓)

## 기준 2 — 재접속률(리텐션) 메커니즘 필수

다시 돌아올 이유를 코드로 설계한다. 아래 메커니즘 중 **최소 2개**를 명시적으로 탑재.

| 메커니즘 | 예시 |
|---|---|
| 스트릭/연속기록 | 끊기면 리셋, 최고기록 보존(손실회피) |
| 매일 갱신 | 날짜 시드로 매일 바뀌는 미션/한마디/팩트 |
| 저장·이어하기 | 진행 저장, 다음 방문에 이어서 |
| 알림 | 리마인드·푸시(PWA) |
| 성장·진화 | 계급/레벨/진화하는 비주얼 |
| 소셜 비교 | 또래·친구·전체 대비 내 위치 |
| 수집 | 배지/마일스톤/컬렉션 채우기 |

### ✅ 체크리스트 (기준 2)
- [ ] (필수) 위 표에서 **2개+** 채택, 각각 어떻게 구현할지 한 줄로 적었다
- [ ] (필수) 최소 1개는 **매일/주기적으로 새 이유**를 만든다 (매일 갱신·스트릭·시즌 중 1+)
- [ ] 첫 방문 후 **24h 내 돌아올 훅**이 무엇인지 말할 수 있다
- [ ] 리텐션 데이터는 **로컬 저장**(가입 없이) 우선, 필요 시 격리스키마 DB

## 기준 3 — 니치 타겟

범용("누구나 쓰는") 금지. **수요는 분명한데 덜 채워진 틈새**를 좁고 강한 의도로 노린다.

판정 공식: **좁은 타겟 + 강한 의도 + 기존 대안의 빈틈**.
아이디어마다 아래 3줄을 반드시 적는다.

### ✅ 체크리스트 (기준 3)
- [ ] (필수) **타겟 한 줄**: 누구인가 (예: "3교대 간호사", "공무원 수험 1년차" — "20대"처럼 넓으면 탈락)
- [ ] (필수) **왜 필요 한 줄**: 그들의 분명한 통증/의도
- [ ] (필수) **기존 대안 빈틈 한 줄**: 지금 뭘 쓰고 있고 뭐가 비어 있나
- [ ] 검색 의도가 분명한 **키워드 1~3개** 댈 수 있다 (SEO 정조준)
- [ ] 팩토리 기존 서비스와 **중복 아님** (projects.json 대조)

## 통과 샘플 아이디어 (세 기준 모두 ✅)

> 각 아이디어는 기준3(타겟/왜/빈틈) + 듀얼모드(개인/그룹) + 리텐션 2+를 충족. 개발 진입 전 projects.json 최종 중복확인 필요.

### 1) 공시생 스터디 리그 📚
- **타겟**: 공무원 시험 1~2년차 장기 수험생 (9급·7급)
- **왜**: 혼자 1년 버티는 고립·이탈이 최대 적, 동료 압박이 절실
- **빈틈**: 기존 스터디플래너(개인 타이머)는 익명 동료 경쟁/실시간 "지금 공부 중" 압박 레이어가 없음
- **개인**: 과목별 진도·누적 공부시간·시험 D-day, 공부 스트릭
- **그룹**: 익명 스터디방 주간 공부시간 랭킹, "지금 공부 중 N명" 실시간 카운터, 1:1 라이벌 대결
- **리텐션**: 스트릭 + 매일 갱신 랭킹 + D-day 카운트다운

### 2) 아기 통잠 트래커 + 또래방 🍼
- **타겟**: 0~12개월 신생아 부모 (밤중 수유로 못 자는)
- **왜**: "우리 애만 안 자나" 불안 + 수면교육 막막함, 또래 비교가 가장 절실
- **빈틈**: 예방접종/이유식 앱은 많지만 *수면교육 진행 기록 + 같은 월령 또래 통잠률 비교*는 빔
- **개인**: 밤잠/낮잠 기록, 통잠(밤새 안 깸) 진행 그래프, 성장 타임라인
- **그룹**: 같은 월령 부모 익명방, "또래 통잠 성공률 %" 비교, 원탭 응원
- **리텐션**: 매일 수면 기록 + 또래 소셜비교 + 통잠 마일스톤 배지(수집)

### 3) 반려동물 다이어트 챌린지 🐶
- **타겟**: 수의사에게 "살 빼라" 들은 비만 반려견·반려묘 보호자
- **왜**: 체중감량은 몇 달짜리 장기전 → 동기·기록이 끊기면 실패
- **빈틈**: 사료계산기·나이계산기는 있어도 *목표 체중까지 추적 + 견종별 다이어트 랭킹*은 없음
- **개인**: 주간 체중 기록·목표 그래프, 권장 칼로리 가이드, 비포/애프터 사진 컬렉션
- **그룹**: 같은 견종/체급 감량 랭킹, 비포애프터 공유 카드, 다이어트 챌린지 시즌
- **리텐션**: 주간 체중 갱신 + 랭킹 소셜비교 + 비포애프터 수집

### 4) 금연 D-DAY · 같이 끊기 🚭
- **타겟**: 금연 N일차 도전자 (특히 재도전·실패 반복자)
- **왜**: 충동은 90초~20분 파도, 그 순간 버틸 도구 + 혼자가 아니란 감각이 성패를 가름
- **빈틈**: 단순 카운터 앱은 많지만 *충동 SOS 대처 + 같이 끊는 익명 동료 카운터*를 묶은 건 드뭄 (검증된 [리부트] 템플릿의 금연 니치 확장)
- **개인**: 금연 스트릭, 절약 금액·되찾은 수명 실시간 계산, 충동 SOS(호흡·대체행동)
- **그룹**: "오늘 함께 버틴 N명" 집단 카운터, 다짐/승리 익명 피드, 스트릭 공유
- **리텐션**: 스트릭 + 매일 절약누적 갱신 + 마일스톤(1·3·7·30·90일) 수집

### 5) 러닝 크루 가상 릴레이 🏃
- **타겟**: 혼자 뛰는 직장인 러너 (러닝크루 모임은 부담스러운)
- **왜**: 혼자선 동기 떨어짐, 그러나 오프라인 크루는 시간·장소 부담 → 비동기 그룹 챌린지 수요
- **빈틈**: 페이스 계산기·만보기는 개인 기록뿐, *익명 크루가 거리를 합산해 목표(예: 서울→부산 422km)를 함께 채우는* 비동기 릴레이는 빔
- **개인**: 러닝 거리·페이스 기록, 개인 누적거리, 주간 목표
- **그룹**: 익명 크루 누적거리 합산 릴레이("우리가 지금 대전 통과!"), 크루 랭킹, 기여도
- **리텐션**: 주간 목표 갱신 + 크루 진행 소셜비교 + 구간 통과 배지(수집)

## 빠른 기획 게이트 (요약 — 아이디어마다 복붙해서 채움)

```
아이디어:
[기준3] 타겟 / 왜필요 / 기존빈틈 / 키워드:
[기준1] 개인레이어: __  | 그룹레이어(상호작용): __  | 개인↔그룹 연결: __
[기준2] 리텐션 메커니즘 2+: __ , __  (1+는 매일/주기 갱신)
[중복확인] projects.json 대조: 통과/유사·중복
판정: 기준1 ✅/❌ · 기준2 ✅/❌ · 기준3 ✅/❌ → 통과/반려
```
