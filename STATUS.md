# VIRAL 앱 진행 보드 (정적 스택 적응판)

원문 설계(`VIRAL-10` Next.js/Supabase 멀티에이전트 프롬프트)를 이 저장소의 실제
스택(**정적 HTML + `gen-pages.mjs`**)에 맞게 적응해 진행. 백엔드/유료 키가 필요한
부분은 소유자 승인 전까지 보류(사유는 CLAUDE.md 참고).
