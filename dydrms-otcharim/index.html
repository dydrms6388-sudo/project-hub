오늘옷장 (기온별 옷차림) 소개

오늘 몇 도, 뭐 입지? 위치만 켜면 오늘 날씨에 딱 맞는 옷차림을 상·하의·아우터·신발·액세서리 한 카드로 대신 골라주는 무료 웹 도구. 널리 통용되는 기온별 옷차림표 8구간을 기준으로, ①기상청 겨울철 체감온도 공식(바람 반영)으로 실제보다 추운 '체감'을 계산해 더 따뜻하게 ②아침·저녁 일교차가 8°C 이상이면 '겹쳐입기(레이어링)' 경고와 처방 ③'추위/더위 잘 탐' 개인 체감 슬라이더(-3~+3)로 한 단계 보정 ④성인(남/여)·아이 대상 + 캐주얼/오피스/액티브 스타일 톤별 코디까지 개인화. Open-Meteo(무료·무키) 지오로케이션으로 현재 기온·풍속·습도·일교차 자동 조회(실패 시 슬라이더·숫자·프리셋 수동 폴백, 접근성 이중화). 재방문 훅=오늘 뭐 입었지 코디 일지(추웠다/딱좋았다/더웠다 기록→다음 추천 자동 보정, 🔥연속 스트릭·최근 14일 기록·'어제보다 N도' 비교). 무백엔드·무가입·무유료API, 위치·기록은 기기 localStorage에만 저장(서버 전송 0). colorfit(퍼스널컬러)과 축분리된 '기온 기반 옷차림' 전용 신규 버티컬(허브 dedup 0). 큐 전부 금융→비금융 발산 선정.
