오라 (Aura)

매일 새로운 성격·연애·감성 심리 테스트를 모은 무료 테스트 플랫폼. 핵심 차별점은 결과를 세상에 하나뿐인 살아 움직이는 3D 오라로 시각화하는 것 — 유형별 색상(hue)·왜곡·입자 파라미터로 react-three-fiber MeshDistort 블롭과 가산혼합 파티클 오라를 실시간 렌더한다. 6종 테스트(나의 연애 색깔 6색, 영혼 동물 6종, 감정 원소 불·물·바람·흙·빛·어둠, 연애 페르소나 6유형, 진짜 매력 포인트 6종, 우주 행성 유형 7행성), 각 8문항·가중치 합산 스코어링으로 38개 결과 유형. 결과 화면엔 3D 오라+성격 해설+특성 해시태그+잘맞는/조심할 오라 궁합+오라의 조언+희귀도 배지. 재방문 훅: 날짜 시드 오늘의 테스트, 연속 방문 스트릭, 수집한 오라 컬렉션(38종 도감), 최근 결과 목록. 바이럴: 결과별 고유 URL(/r/슬러그/유형)과 동적 OG 1200x630 공유카드(Noto Sans KR 서브셋 임베드), Web Share. 몰입 4종: r3f 3D 오라(디스토트 블롭+입자), framer-motion 문항 전환·진행바·결과 스태거·계산 연출, canvas-confetti(결과 공개), Web Audio 합성 사운드(탭·선택·상승 차임·앰비언트 패드, 음소거 기본 OFF). 무DB 전 클라이언트(기록·스트릭·컬렉션 localStorage, 서버 전송 없음).
