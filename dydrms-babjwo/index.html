밥줬어? 소개

밥줬어?는 가족이 함께 반려동물의 밥, 약, 물, 산책을 관리하는 공유 체크 도구입니다. 회원가입 없이 6자리 코드나 링크를 공유하면 가족 모두의 스마트폰에서 같은 화면을 실시간으로 볼 수 있어 '오늘 밥 줬어?'라는 질문이 필요 없어집니다. 매일 자정에 자동으로 초기화되어 오늘 아직 완료하지 않은 항목을 쉽게 확인할 수 있습니다.

이렇게 사용하세요

  1. 첫 화면에서 반려동물 이름과 관리 항목(밥·약·물·산책)을 설정합니다.
  2. 생성된 6자리 코드 또는 공유 링크를 가족 단톡방이나 문자로 전달합니다.
  3. 가족 누구든 항목을 완료하면 체크 버튼을 눌러 기록합니다.
  4. 다른 가족의 화면에 실시간으로 완료 여부가 표시되는지 확인합니다.
  5. 정기 투약 기록은 최근 7일 차트에서 누가 언제 줬는지 확인합니다.

활용 팁

  • 하루에 2회 급여한다면 항목을 아침·저녁으로 나눠 설정하면 누락 방지에 더 효과적입니다.
  • 심장사상충약 같은 월 1회 투약은 투약 후 날짜를 메모에 남겨두면 다음 달 일정을 놓치지 않습니다.
  • 여러 마리를 키운다면 각 반려동물을 별도 항목으로 등록해두면 관리가 훨씬 편해집니다.

자주 묻는 질문

Q. 코드를 잃어버리면 기록도 사라지나요?

코드 또는 링크를 다시 찾지 못하면 해당 방에 재접속하기 어렵습니다. 링크를 북마크하거나 가족 공유 메모에 저장해두는 것을 권장합니다.

Q. 매일 자정 초기화 외에 수동으로 초기화할 수 있나요?

현재는 매일 자정(KST) 자동 초기화만 지원합니다. 투약처럼 매일 반복되지 않는 항목은 항목 설명에 메모를 남겨 구분하면 됩니다.

Q. 인터넷 연결이 없으면 다른 가족 화면에도 바로 반영되나요?

실시간 동기화는 인터넷 연결이 필요합니다. 연결이 끊겼다가 다시 연결되면 자동으로 최신 상태로 업데이트됩니다.
